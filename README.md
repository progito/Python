### 2.1 Синтаксис Python

Синтаксис Python отличается своей читаемостью и простотой, что делает язык более доступным для новичков. Вот некоторые ключевые особенности:

**Отступы:** Во многих языках программирования отступы (пробелы и табуляции в начале строки кода) используются только для улучшения читаемости. Однако в Python отступы имеют синтаксическое значение. Код, который должен быть выполнен внутри функции, цикла, условного оператора или любого другого блока кода, должен быть с отступом.

```python

for i in range(5):
    print(i)  # Эта строка с отступом, поэтому она выполняется в цикле
print("Done")  # Эта строка без отступа, поэтому она выполняется после цикла
```

**Что такое print() и for.. и range..??**

**print()** - это такая функция которая используется для вывода какой либо информации в консоль. Например самая популярная фраза Hello, world! можно вывести так:

```python
print("Hello, world!") # Hello, world!
```
или вот еще пример для вывода числа
```python
print(123) # 123
```

**for** -это один из видов циклов в языке Python. **Цикл** - это разновидность управляющей конструкции в высокоуровневых языках программирования, предназначенная для организации многократного исполнения набора инструкций. (более подробно познакомимся с ними чуть позже)

**Ну а что же такое range..??**

**range()** - она генерирует список чисел, который обычно используется для работы с циклом for. (со списками мы тоже чуть позже разберемся - это как массив) 
```python
range(1, 4) # список от 1 до 4 [1, 2, 3, 4]
```
если первый аргумент не указан тогда последовательность автоматически начинается с 0. т.е 0, 1, 2...

**Колонки:** В Python двоеточие (:) используется для начала нового блока кода (например, после определения функции, цикла или условного оператора).

```python

def my_function():  # Колонка указывает на начало блока кода функции
    print("Hello, World!")
```

### Переменные и типы данных

**Переменные в Python** представляют собой имена, которым мы присваиваем значения. Это могут быть числа, строки, списки, объекты и многое другое. В Python переменные не требуют явного объявления типа, как в некоторых других языках программирования. Вы просто присваиваете значение переменной, и Python определит тип данных автоматически.

```python
x = 5  # Создаем переменную x и присваиваем ей значение 5
y = "Hello, World!"  # Создаем переменную y и присваиваем ей строку
```

**Типы данных** определяют, какие операции можно выполнять с данными. Python имеет множество встроенных типов данных. Вот некоторые из них:

**Целые числа (Integers):** Это числа без десятичной точки. Пример: 5, 100, -10

```python
x = 10  # x теперь целое число
```

**Числа с плавающей точкой (Floats):** Это числа с десятичной точкой. Пример: 5.0, 1.23, -3.14

```python
y = 7.5  # y теперь число с плавающей точкой
```

**Строки (Strings):** Строки используются для работы с текстом. Пример: "Hello", 'World'

```python
name = "Alice"  # name теперь строка
```

**Булевы значения (Booleans):** Булевы значения могут быть либо True (истина), либо False (ложь) и используются в условных выражениях и циклах.

```python
is_true = False  # is_true теперь булевая переменная
```

+ еще есть списки, словари и тп..
  
### 2.3 Операторы

**Операторы** - это специальные символы, которые выполняют определенные операции между переменными и значениями. В Python операторы делятся на несколько типов:

**Арифметические операторы:** Они используются для выполнения математических операций, таких как сложение, вычитание, умножение и т.д.

Например:

```python
a = 10
b = 20
print(a + b)  # 30
print(a - b)  # -10
print(a * b)  # 200
print(a / b)  # 0.5
```

**Сравнительные (или отношения) операторы:** Они используются для сравнения значений. Например: ==, !=, >, <, >=, <=.

Например:
```python
a = 10
b = 20
print(a == b)  # False
print(a != b)  # True
print(a < b)   # True
print(a > b)   # False
```

**Логические операторы:** Они используются для выполнения логических операций, таких как "и", "или" и "не".

Например:
```python
a = True
b = False
print(a and b)  # False
print(a or b)   # True
print(not a)    # False
```

**Операторы присваивания:** Они используются для присваивания значений переменным. Например: =, +=, -=, *=, /= и так далее.

Например:
```python
a = 10    # присваивание
a += 5    # эквивалентно a = a + 5
print(a)  # 15
```

**Битовые операторы:** Они используются для работы с битами данных. Например: & (логическое И), | (логическое ИЛИ), ^ (Исключающее ИЛИ), ~ (отрицание), << (сдвиг влево), >> (сдвиг вправо).

Например:
```python
a = 10   # бинарное: 1010
b = 4    # бинарное: 0100
print(a & b)  # 0  (бинарное: 0000)
print(a | b)  # 14 (бинарное: 1110)
print(a << b) # 160 (бинарное: 10100000)
```

**Операторы членства:** Они используются для проверки, является ли значение частью последовательности, например, списка, строки, кортежа и т.д. Они включают в себя in и not in.

Например:
```python
my_list = [1, 2, 3, 4, 5]
print(3 in my_list)     # True
print(6 not in my_list) # True
```

**Операторы тождественности:** Они используются для сравнения объектов, не их значений. Они включают is и is not.

Например:
```python
a = [1, 2, 3] # массив (список)
b = a  # b ссылается на тот же объект, что и a
c = a.copy()  # c является отдельным объектом, хотя и содержит те же значения
print(b is a)  # True
print(c is a)  # False
print(c is not a)  # True
```

Таким образом, операторы в Python позволяют нам выполнять множество различных операций, будь то математические вычисления, сравнения, манипуляции с битами или проверка вхождения элементов в коллекции.

### 2.4 Условные операторы и циклы

**2.4.1 Условные операторы:** Условные операторы в Python используются для выполнения определенных действий в зависимости от условий. Основные условные операторы в Python - это if, elif, else.

```python
age = 18
if age >= 18:
    print("Вы можете голосовать.")
elif age >= 16:
    print("Вы скоро сможете голосовать.")
else:
    print("Вы еще не можете голосовать.")
```

В этом примере, если возраст больше или равен 18, программа выведет "Вы можете голосовать". Если возраст между 16 и 18, программа выведет "Вы скоро сможете голосовать". В любом другом случае, программа выведет "Вы еще не можете голосовать".

**2.4.2 Циклы:** Циклы в Python используются для повторения определенного блока кода. Основные типы циклов в Python - это for и while.

Цикл for обычно используется для перебора элементов в коллекции (например, списка) или для выполнения действия определенное количество раз.

```python
for i in range(5):
    print(i)
```
В этом примере код внутри цикла for будет выполнен 5 раз, и при каждом выполнении будет печататься значение переменной i.

Цикл while используется для выполнения блока кода до тех пор, пока условие истинно.

```python
i = 0
while i < 5:
    print(i)
    i += 1
```

В этом примере код внутри цикла while будет выполнен до тех пор, пока i меньше 5. На каждом проходе цикла i увеличивается на 1.

Это базовое понимание условных операторов и циклов в Python. Они являются основными элементами логики программы и контроля потока в вашем коде.

### 2.5 Работа со строками и файлами

**2.5.1 Работа со строками:** Python предоставляет множество функций и методов для работы со строками. Они позволяют выполнять манипуляции со строками, такие как конкатенация (объединение), сравнение, поиск и замена подстрок, преобразование регистра и т.д.

```python
s = "Hello, World!"
print(s.lower())  # hello, world!
print(s.upper())  # HELLO, WORLD!
print(s.replace('World', 'Python'))  # Hello, Python!
```

**2.5.2 Работа с файлами:** Python предоставляет встроенные функции для чтения и записи файлов. Вы можете открыть файл с помощью функции open(), прочитать его содержимое с помощью метода read(), записать данные в файл с помощью метода write(), и закрыть файл с помощью метода close().

```python
f = open("myfile.txt", "w")  # открываем файл для записи
f.write("Hello, World!")  # записываем строку в файл
f.close()  # закрываем файл

f = open("myfile.txt", "r")  # открываем файл для чтения
print(f.read())  # читаем и печатаем содержимое файла
f.close()  # закрываем файл
```

В Python есть также контекстные менеджеры (with statement), которые автоматически закрывают файлы, даже если происходит ошибка.

```python
with open("myfile.txt", "r") as f:
    print(f.read())
```

В этом примере файл будет автоматически закрыт после выхода из блока with, и вам не нужно вызывать f.close().

Важно знать, что работа со строками и файлами - это базовые навыки при работе с Python, поскольку они используются во многих приложениях и задачах обработки данных.

### 2.7 правила PEP
PEP в Python означает Python Enhancement Proposal. PEP - это документ, который описывает новые функции Python, процессы и окружения, которые влияют на большую часть сообщества Python.

Однако, когда говорят о "правилах PEP", часто имеют в виду PEP 8 - это документ, который описывает стандарты кодирования в Python для улучшения читаемости и согласованности кода.

Вот некоторые основные рекомендации из PEP 8:

**Именование:** Используйте CamelCase для имен классов, lowercase_with_underscores для функций и методов, и UPPER_CASE_WITH_UNDERSCORES для констант.

```python
# Пример:
class MyFirstClass:
    def my_function():
        MY_CONSTANT = 10
```

**Отступы:** Используйте 4 пробела на уровень отступа, а не табуляцию.

```python
# Пример:
def my_function():
    if True:
        return "Hello, World!"
```

**Длина строки:** Строки должны быть не более 79 символов.

**Пробелы:** Используйте пробелы вокруг операторов и после запятой для улучшения читаемости.

```python
# Пример:
my_variable = my_function(arg1, arg2)
```

**Импорты:** Импорты должны быть на отдельных строках и размещаться в начале файла.

```python
# Пример:
import os
import sys
```
Это лишь некоторые из рекомендаций PEP 8. Цель PEP 8 - сделать код Python более читаемым и понятным для людей.


### 2.8 F-строки

**F-строки** (форматированные строковые литералы) в Python - это способ форматирования строк, который был введен в Python 3.6. Они начинаются с f или F перед самой строкой и имеют фигурные скобки {} содержащие выражения, которые будут заменены их значениями.

**Преимущества использования f-строк включают:**

    1. Синтаксическую простоту.
    2. Возможность вставки любого допустимого Python-выражения внутрь фигурных скобок.
    3. Они могут быть более читабельными и удобными для пользователей, поскольку вставленные переменные или выражения можно видеть напрямую в строке.

**Пример 1:**
```python
name = "Alice"
print(f"Hello, {name}!")
```
Вывод: Hello, Alice!

Здесь {name} внутри f-строки заменяется на значение переменной name.

**Пример 2:**

Вы можете использовать f-строки для выполнения вычислений прямо внутри фигурных скобок.

```python
a = 10
b = 20
print(f"Sum: {a + b}")
```
Вывод: Sum: 30

**Пример 3:**

F-строки также могут работать с функциями и методами.

```python
name = "alice"
print(f"Hello, {name.capitalize()}!")
```
Вывод: Hello, Alice!

В этом примере мы используем метод .capitalize() прямо внутри фигурных скобок, чтобы преобразовать первую букву имени в заглавную.

**Пример 4:**

С помощью f-строки можно форматировать числа. Например, можно округлить число до определенного количества знаков после запятой.

```python
pi = 3.1415926
print(f"Pi rounded to two decimal places: {pi:.2f}")
```
Вывод: Pi rounded to two decimal places: 3.14

Здесь .2f внутри фигурных скобок говорит Python округлить число до двух знаков после запятой.
